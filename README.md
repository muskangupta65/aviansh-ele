Avinash Electronics Store Website

A responsive electronics store website built using HTML, CSS, and JavaScript.
This project showcases a modern UI design for an online electronics shop.

📌 Project Description
Avinash Electronics Store is a front-end web project designed to simulate an online electronics shopping experience. The website includes product listings, navigation, and interactive UI components.
This project was created for learning and practice purposes.

🚀 Features
✅ Responsive design (works on mobile & desktop)
✅ Clean and modern UI
✅ Product showcase section
✅ Navigation bar
✅ Interactive elements using JavaScript
✅ Easy to customize

🛠️ Technologies Used
HTML5 – Structure of the website
CSS3 – Styling and layout
JavaScript – Interactivity and dynamic behavior
